ifconfig 命令详解

发表于 2016-01-18   |   分类于 linux

定义和用法

ifconfig命令常用来显示,配置系统中的网络接口（网卡）信息，如激活、关闭、设置地址等。

在Linux系统中，网卡命名规律：eth0为第一块以太网卡（Ethernet Card），eth1为第二块。

语法

1. ifconfig [网络设备] [省略x个参数]

例子 1

ifconfig 如果不接任何参数，就会输出当前网络接口的情况

1. [root@localhost ~]\# ifconfig

运行结果：

1. eth0
2. Link encap:Ethernet HWaddr 00:C0:9F:94:78:0E
3. inet addr:192.168.1.88 Bcast:192.168.1.255 Mask:255.255.255.0
4. inet6 addr: fe80::2c0:9fff:fe94:780e/64 Scope:Link
5. UP BROADCAST RUNNING MULTICAST MTU:1500 Metric:1
6. RX packets:850 errors:0 dropped:0 overruns:0 frame:0
7. TX packets:628 errors:0 dropped:0 overruns:0 carrier:0
8. collisions:0 txqueuelen:1000
9. RX bytes:369135 (360.4 KiB) TX bytes:75945 (74.1 KiB)
10. Interrupt:10 Base address:0x3000

用ifconfig 来调试 eth0网卡的地址

1. [root@localhost ~]\# ifconfig eth0 down
2. [root@localhost ~]\# ifconfig eth0 192.168.1.99 broadcast 192.168.1.255 netmask 255.255.255.0
3. [root@localhost ~]\# ifconfig eth0 up
4. [root@localhost ~]\# ifconfig eth0

上例解释：

第一行：ifconfig eth0 down 表示如果eth0是激活的，就把它DOWN掉。此命令等同于 ifdown eth0；
第二行：用ifconfig 来配置 eth0的IP地址、广播地址和网络掩码；
第三行：用ifconfig eth0 up 来激活eth0 ； 此命令等同于 ifup eth0
第四行：用 ifconfig eth0 来查看 eth0的状态；
