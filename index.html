linux 免密码SSH登陆设置

发表于 2016-01-18   |   分类于 linux

作用：可以在不需要密码的情况下，实现一些自动化任务

原理：将客户机的公钥加入到服务器的信任列表里面，客户机使用私钥加密传输数据，服务器使用公钥解密数据

需要用到的密令：

  • ssh-keygen 生成ssh公钥
  • ssh-copy-id 将本地公钥拷贝到远程的authorized keys

用到的机器：

  1. 客户机(192.168.0.10) 保存私钥
  2. 服务器(192.168.0.11) 保存公钥

过程：

  1. 在192.168.0.10（客户机）上使用ssh-keygen 生成密码对
  2. 在192.168.0.10（客户机）上使用ssh-copy-id 将公钥上传到192.168.0.11（服务器）

ifconfig 命令详解

发表于 2016-01-18   |   分类于 linux

定义和用法

ifconfig命令常用来显示,配置系统中的网络接口（网卡）信息，如激活、关闭、设置地址等。

在Linux系统中，网卡命名规律：eth0为第一块以太网卡（Ethernet Card），eth1为第二块。

语法

1. ifconfig [网络设备] [省略x个参数]

例子 1

ifconfig 如果不接任何参数，就会输出当前网络接口的情况

1. [root@localhost ~]\# ifconfig

运行结果：

1. eth0
2. Link encap:Ethernet HWaddr 00:C0:9F:94:78:0E
3. inet addr:192.168.1.88 Bcast:192.168.1.255 Mask:255.255.255.0
4. inet6 addr: fe80::2c0:9fff:fe94:780e/64 Scope:Link
5. UP BROADCAST RUNNING MULTICAST MTU:1500 Metric:1
6. RX packets:850 errors:0 dropped:0 overruns:0 frame:0
7. TX packets:628 errors:0 dropped:0 overruns:0 carrier:0
8. collisions:0 txqueuelen:1000
9. RX bytes:369135 (360.4 KiB) TX bytes:75945 (74.1 KiB)
10. Interrupt:10 Base address:0x3000

用ifconfig 来调试 eth0网卡的地址

1. [root@localhost ~]\# ifconfig eth0 down
2. [root@localhost ~]\# ifconfig eth0 192.168.1.99 broadcast 192.168.1.255 netmask 255.255.255.0
3. [root@localhost ~]\# ifconfig eth0 up
4. [root@localhost ~]\# ifconfig eth0

上例解释：

第一行：ifconfig eth0 down 表示如果eth0是激活的，就把它DOWN掉。此命令等同于 ifdown eth0；
第二行：用ifconfig 来配置 eth0的IP地址、广播地址和网络掩码；
第三行：用ifconfig eth0 up 来激活eth0 ； 此命令等同于 ifup eth0
第四行：用 ifconfig eth0 来查看 eth0的状态；

ping 命令详解

发表于 2016-01-18   |   分类于 linux

定义和用法

ping命令是常用的网络命令，通常用来测试与目标主机的连通性。

语法

1. ping [参数] [主机名或IP地址]

参数描述-dSocket SO_DEBUG功能。-f极限检测。大量且快速地发送网络封包给一台机器，看它的回应。-n只输出数值。-q不显示任何传送封包的信息，只显示最后的结果。-r忽略普通的Routing Table，直接将数据包送到远端主机上。通常是查看本机的网络接口是否有问题。-R记录路由过程。-v详细显示指令的执行过程。\-c数目：在发送指定数目的包后停止。-i秒数：设定间隔几秒送一个网络封包给一台机器，预设值是一秒送一次。-I网络界面：使用指定的网络界面送出数据包。-l前置载入：设置在送出要求信息之前，先行发出的数据包。-p范本样式：设置填满数据包的范本样式。-s字节数：指定发送的数据字节数，预设值是56，加上8字节的ICMP头，一共是64ICMP数据字节。-t存活数值：设置存活数值TTL的大小。

例子

1. \# 最普遍用法
2. ping 192.168.120.205
3. \# ping网关
4. ping -b 192.168.120.1
5. \# ping指定次数
6. ping -c 10 192.168.120.206
7. \# 时间间隔和次数限制的ping
8. ping -c 10 -i 0.5 192.168.120.206
9. \# 通过域名ping公网上的站点
10. ping -c 5 www.58.com
11. \# 多参数使用
12. \# -i 3 发送周期为 3秒 -s 设置发送包的大小为1024
13. \# -t 设置TTL值为 255
14. ping -i 3 -s 1024 -t 255 192.168.120.206

netstat 命令详解

发表于 2016-01-18   |   分类于 linux

定义和用法

用于显示各种网络相关信息，如网络连接，路由表，接口状态 (Interface Statistics)，masquerade 连接，多播成员 (Multicast Memberships) 等等

语法

1. netstat [option]

参数描述-a(all)显示所有选项，默认不显示LISTEN相关-t(tcp)仅显示tcp相关选项-u(udp)仅显示udp相关选项-n拒绝显示别名，能显示数字的全部转化成数字。-l仅列出有在 Listen (监听) 的服務状态-p显示建立相关链接的程序名-c每隔一个固定时间，执行该netstat命令。-r显示路由信息，路由表-e显示扩展信息，例如uid等-s按各个协议进行统计

例子

1. \# 常用
2. netstat -ntlp
3. \#列出所有端口
4. netstat -a
5. \#列出所有 tcp 端口
6. netstat -at
7. \#列出所有 udp 端口
8. netstat -au
9. \#只显示监听端口
10. netstat -l
11. \#只列出所有监听 tcp 端口
12. netstat -lt
